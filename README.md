auto intl

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-cn-intl', options],
  ],
}
```

```js
"scripts": {
  "i18n": "umi cnintl src/pages/test"
},
```
